# Politica de Privacidade

> Modelo inicial para adaptar com seus dados juridicos. Nao substitui revisao juridica.

## 1. Quem somos

Este aplicativo e operado por `Quaraci Dev`, contato em `Staffquaracidev@gmail.com`.

## 2. Dados coletados

Podemos coletar:

- imagens enviadas para gerar diagramas
- dados tecnicos de sessao e uso
- rascunhos e historico local do proprio app
- dados minimos da conta Google e backups, apenas quando o usuario autorizar a sincronizacao

## 3. Finalidade do uso

Os dados sao usados para:

- gerar diagramas de croche a partir das imagens
- manter o funcionamento e a seguranca operacional do app
- restaurar configuracoes e resultados em backups opcionais
- prestar suporte quando necessario

## 4. Compartilhamento

Podemos compartilhar dados com operadores necessarios para funcionamento do servico, como:

- provedor de IA
- provedor de hospedagem e banco de dados
- Google Drive, apenas quando o usuario ativar o backup

## 5. Retencao

Os dados ficam armazenados pelo tempo necessario para operacao do servico, cumprimento legal e resolucao de disputas. Backups no Google podem ser removidos pelo proprio usuario.

## 6. Direitos do usuario

O usuario pode solicitar:

- acesso aos dados
- correcao
- exclusao, quando aplicavel
- informacoes sobre tratamento

Solicitacoes devem ser enviadas para `Staffquaracidev@gmail.com`.

## 7. Seguranca

Adotamos medidas razoaveis de seguranca, mas nenhum sistema e absolutamente invulneravel.

## 8. Menores de idade

O servico nao e direcionado intencionalmente a menores sem supervisao legal adequada.

## 9. Alteracoes

Esta politica pode ser atualizada. A versao vigente ficara disponivel no endereco oficial do aplicativo.

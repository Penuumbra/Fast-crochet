Politica de Privacidade

Operadora: Quaraci Dev

Contato: Staffquaracidev@gmail.com

1. Dados coletados

Podemos tratar imagens enviadas para gerar diagramas, dados tecnicos basicos de uso, rascunhos locais e, quando o usuario optar por conectar o Google, dados minimos da conta e backups gravados na pasta privada do app no Google Drive.

2. Finalidade

Os dados sao usados para gerar diagramas de crochet, manter o funcionamento do app, restaurar preferencias e permitir backup opcional entre aparelhos.

3. Compartilhamento

Podemos compartilhar dados com provedores necessarios ao funcionamento do servico, como OpenAI para geracao, Render e Neon para hospedagem e persistencia, e Google apenas quando o usuario escolhe ativar o backup.

4. Retencao

Os dados sao mantidos pelo tempo necessario para a operacao do servico, cumprimento legal e resolucao de disputas. Backups no Google Drive podem ser apagados pelo proprio usuario ao desconectar ou remover os dados do app.

5. Direitos do usuario

O usuario pode solicitar acesso, correcao ou exclusao de dados aplicaveis pelo e-mail Staffquaracidev@gmail.com.

6. Seguranca

Adotamos medidas razoaveis de seguranca, embora nenhum sistema seja absolutamente invulneravel.

7. Alteracoes

Esta politica pode ser atualizada. A versao vigente ficara disponivel nesta pagina.

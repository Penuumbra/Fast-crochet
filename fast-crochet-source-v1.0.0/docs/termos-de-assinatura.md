# Termos da Assinatura

> Modelo inicial para adaptar com seus dados juridicos. Nao substitui revisao juridica.

## 1. Plano

- nome: `Plano Essencial`
- preco: `R$ 3,69 por mes`
- recorrencia: mensal

## 2. Trial

- o teste gratuito tem duracao de `7 dias`
- durante esse periodo nao ha obrigacao de cadastro
- apos o trial, o uso fica bloqueado ate login/cadastro e assinatura

## 3. Cobranca

- a cobranca passa a valer somente quando o usuario contrata o plano pago
- o ciclo de renovacao e mensal, salvo cancelamento

## 4. Cancelamento

- o usuario pode cancelar a renovacao futura conforme o meio de pagamento utilizado
- o cancelamento impede novas renovacoes, mas o acesso pode permanecer ate o fim do periodo ja pago

## 5. Reajustes

O valor da assinatura pode ser alterado futuramente mediante comunicacao previa, conforme legislacao aplicavel e regras do meio de pagamento.

## 6. Inadimplencia

Se houver falha de pagamento, o acesso pode ser suspenso ou encerrado ate regularizacao.
